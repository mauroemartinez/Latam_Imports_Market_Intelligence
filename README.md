*Market Intelligence solution designed to bridge trade gaps between Italian and Latin American companies through data-driven insights.*

- **Data Engineering:** Automated end-to-end pipelines integrating internal sources with **Softrade** trade databases.
- **Web Deployment:** Led the embedding of Power BI reports into the company’s website, implementing **Role-Level Security (RLS)** for secure partner access.
- **Lead Gen Logic:** Engineered data cleansing and transformation workflows to identify high-potential B2B importers.
- **Strategic Reporting:** Implemented an "Actionable Bullets" system to deliver concise, metric-based summaries alongside visual dashboards.
